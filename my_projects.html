PY_JUNE

Forum for IT specialists which was created using technologies such as Python, HTML, JavaScript, CSS, PostgreSQL and on Django framework. The goal of this project was to use skills and improve them. The project includes registration using third-party services such as Google and GitHub, and it is also possible to customize the profile. The main features of this project are the ability to create/edit/delete posts in any topic. Any post or comment can be rated with like. It is also possible to leave comments on posts or to other comments. One of the features of the project is user translation.
